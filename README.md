## infernojs mobile demo

> 使用了类react框架 inferno.js，以及ant-mobile.
> 借助于官方脚手架 create

[infernojs](https://infernojs.org/)

- 官方实例

```
npx create-inferno-app my-app
cd my-app
npm start
```
- 提取官方配置，加入适合自己项目开发的一些配置

1. 安装依赖 `yarn`
2. 开发运行 `yarn start`
3. 构建 `yarn build`
4. 模拟部署  `yarn serve`
> 别忘记 `yarn global add serve`
